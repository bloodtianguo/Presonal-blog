基于vue实现的 **个人主页** 模板
<!-- 👉[在线预览](https://cv.manerfan.com)👈  -->
## 特性
- 屏幕自适应
- Banner模块
- 关于模块
- 履历模块
- 博客模块（RSS订阅）
- 菜单浮层（小屏）
- json文件配置（修改模块内容，模块顺序）
- markdown
- 纯前端工程
- 可扩展（自定义模块）
